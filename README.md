## 🚀 О проекте

Это максимально оптимизированное кастомное GKI-ядро для **Xiaomi 14T Pro (`rothko`)**. Главная цель проекта — выжать максимум производительности из флагманского железа, при этом значительно улучшив энергоэффективность в режиме ожидания. 

Ядро собирается с использованием **LTO=full** (Link-Time Optimization) для достижения эталонной скорости работы процессорных инструкций, а в качестве Root-доступа используется **KernelSU-Next + SUSFS** (для идеального скрытия рута от банковских приложений).

## ✨ Ключевые особенности и твики

### 🔒 Root и Безопасность
- **KernelSU-Next (dev-susfs):** Рут-доступ прямо в ядре без модификации системных разделов.
- **SUSFS v2.1.0+:** Мощнейший анти-детект. Полное скрытие `su`, точек монтирования и фейковых данных о системе для банков и игр.
- **Baseband Guard (BBG):** Защита критически важных разделов модема от перезаписи.
- **DroidSpaces-OSS:** Улучшенная поддержка изоляции процессов.

### ⚡ Производительность (Эксклюзив)
- **LTO=full (Full Link-Time Optimization):** Ядро компилируется с максимальной глобальной оптимизацией. Это увеличивает производительность CPU на 3-5% и уменьшает размер бинарника.
- **UFS 4.0 Turbo I/O:** Буфер `read_ahead_kb` увеличен до **1024 КБ**. Тяжелые игры, локальные видео и галерея прогружаются мгновенно благодаря чтению огромными блоками, что заодно позволяет контроллеру памяти быстрее уйти в сон.

### 🔋 Автономность и Память
- **Ultra Battery Saver (Sysctl Tuning):** Ядерный логгер (`printk`) отключен для исключения фоновых пробуждений CPU. Дисковые кэши (`vm.dirty_ratio`) настроены на редкую, но объемную запись — накопитель больше не просыпается каждую секунду.
- **ZSTD ZRAM:** Алгоритм сжатия оперативной памяти переведен на `zstd`. Он сжимает данные в 3 раза эффективнее стандартного lz4, что позволяет держать десятки приложений в фоне.
- **TEO Idle Governor:** Более умный планировщик сна процессора. Эффективнее сбрасывает частоты ядер в глубоком сне.
- **MGLRU & DAMON:** Современные алгоритмы Google для умной очистки оперативной памяти.

### 🌐 Сеть и Файловые системы
- Включены продвинутые TCP-алгоритмы (BBR, FQ_CODEL) для снижения пинга.
- Встроенная поддержка NTFS3 и ExFAT.

## 📦 Модуль-компаньон (ZSTD & TEO Enabler)

Для удобного управления эксклюзивными фичами (UFS Turbo, Battery Saver, ZSTD, TEO) в репозитории предусмотрен специальный модуль для KernelSU.
Он интегрируется прямо во вкладку "Модули" (WebUI) внутри приложения KernelSU:

1. Скачайте модуль.
2. Прошейте в KernelSU-Next.
3. Перезагрузите телефон и нажмите **Открыть** модуль — внутри вас ждут 4 стильных тумблера для управления системой в реальном времени!

## 🛠 Инструкция по сборке (Self-Hosted Runner)

Сборка ядра с **LTO=full** требует очень много оперативной памяти (от 16 до 32 ГБ RAM + Swap) на этапе линковки. Стандартные бесплатные серверы GitHub Actions (7 ГБ) **не справятся** с этой задачей (ошибка OOM).

Именно поэтому проект настроен для работы на вашем мощном **Self-Hosted сервере**:

1. Подключите ваш VPS (рекомендуется 8 vCPU, 16+ GB RAM) к репозиторию как runner.
2. В Actions выберите `Build Xiaomi 14T Pro Wild KernelSU-Next`.
3. Нажмите `Run workflow`.
4. Скрипт автоматически скачает `Bazel`, исходники Android 14 GKI, создаст файл подкачки на 16 ГБ и запустит хардкорную компиляцию!

*Сборка обычно занимает около 40-60 минут.*
